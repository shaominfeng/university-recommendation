An intelligent platform to help students find their perfect university match

帮助学生找到完美大学匹配的智能平台

📋 Overview

The University Recommendation System is a web-based application designed to help high school students find suitable universities based on their college entrance examination scores. The system provides personalized recommendations by analyzing historical admission data and matching student scores with university admission requirements.

✨ Features

🎯 Score-based Recommendations

Input your exam scores to get personalized university recommendations based on historical data analysis.

📊 Comprehensive Database

Includes admission data for universities in Jiangsu Province with 2021 data available and expanding coverage.

⚡ Real-time Analysis

Automatic calculation and ranking based on your scores with instant results and detailed insights.

🎨 User-friendly Interface

Modern, responsive design built with React and Ant Design for optimal user experience.

🛠️ Tech Stack

Frontend:

React 19.1.0 TypeScript Ant Design 5.25.2 Vite 6.3.5 React Router DOM

Backend:

NestJS 8.0.0 MongoDB Mongoose Node.js TypeScript

📦 Installation

Prerequisites

  • Node.js >= 16.14.2
  • Git
  • MongoDB (for backend)

Quick Start

Clone the repository 
git clone https://github.com/shaominfeng/UniversityRecommend.git
cd university-recommendation
Install Frontend Dependencies 
cd app/frontend
npm install
Install Backend Dependencies 
cd app/app-server
npm install
Start Development Servers

Frontend:

cd app/frontend
npm run dev

Backend:

cd app/app-server
npm run start:dev
Access the Application
  • Frontend: http://localhost:5173
  • Backend API: http://localhost:3000

📋 项目概述

大学推荐系统是一个基于Web的应用程序，旨在帮助高中生根据高考成绩找到合适的大学。系统通过分析历史录取数据，将学生成绩与大学录取要求进行匹配，提供个性化的推荐建议。

✨ 主要功能

🎯 基于分数的推荐

输入高考成绩，基于历史数据分析获得个性化大学推荐。

📊 全面的数据库

包含江苏省各大学录取数据，提供2021年数据并持续扩大覆盖范围。

⚡ 实时分析

基于您的分数自动计算和排名，提供即时结果和详细洞察。

🎨 友好的用户界面

使用React和Ant Design构建的现代响应式设计，提供最佳用户体验。
